Robotics

Creative coding, industrial robots, APIs, and microprocessors.

P5.js

P5.js is a JavaScript library for creative coding in the browser. Students use the online editor to draw, animate, and build interactive sketches — no installation required. The RGB color picker is especially useful for working with the color() function.

  • P5.js
  • P5.js Web Editor
  • RGB Color Picker

FANUC

Students learn to program and operate FANUC industrial robots. Topics include using the teach pendant, working with position registers, and writing basic motion instructions for pick-and-place and path operations.

  • FANUC America

JSON & APIs

Using Daniel Shiffman’s Coding Train tutorials, students learn to fetch live data from web APIs, parse JSON responses, and build sketches that respond to real-world data.

  • The Coding Train

Raspberry Pi Pico

An introduction to microprocessors using the Raspberry Pi Pico — a small, low-cost microcontroller. Students write MicroPython programs to control inputs and outputs, bridging the gap between software and physical hardware.

  • Raspberry Pi Pico
